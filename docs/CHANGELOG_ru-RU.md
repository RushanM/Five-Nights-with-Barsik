# Список изменений

В этом документе представлены изменения в выпусках видеоигры «Пять ночей с Барсиком».

Сначала пишется полное название выпуска с номером версии в изначально представленном версионировании, после, если в начале был написан номер версии в старой системе версионирования, в скобках номер уже в [системе версионирования Дефлекты](https://github.com/RushanM/Deflectas-Versioning), а затем через тире дата объявления выхода выпуска.

* все дальнейшие изменения в этом обновлении будут раскрыты по мере разработки…
